Table: order_items
Schema: SALES
Connection: oracle_10

Description:
Line items for each order, linking orders to products with quantities and pricing.

Columns:
- order_item_id (NUMBER, Primary Key): Unique identifier for each order line
- order_id (NUMBER, Foreign Key → orders.order_id): Reference to parent order
- product_id (NUMBER, Foreign Key → products.product_id): Reference to product
- quantity (NUMBER, NOT NULL): Number of units ordered
- unit_price (NUMBER(10,2), NOT NULL): Price per unit at time of order
- discount_amount (NUMBER(10,2), DEFAULT 0): Discount applied to this line
- line_total (NUMBER(10,2)): Total for this line (calculated)
- tax_amount (NUMBER(10,2), DEFAULT 0): Tax amount for this line

Indexes:
- PK_ORDER_ITEMS on order_item_id
- IDX_ITEM_ORDER on order_id
- IDX_ITEM_PRODUCT on product_id

Foreign Keys:
- FK_ITEM_ORDER: order_id → orders.order_id
- FK_ITEM_PRODUCT: product_id → products.product_id

Related Tables:
- orders (order_id references)
- products (product_id references)

Example Queries:
-- Get all items for a specific order
SELECT oi.*, p.product_name 
FROM order_items oi 
JOIN products p ON oi.product_id = p.product_id 
WHERE oi.order_id = 1001;

-- Find most ordered products
SELECT p.product_name, SUM(oi.quantity) as total_quantity 
FROM order_items oi 
JOIN products p ON oi.product_id = p.product_id 
GROUP BY p.product_name 
ORDER BY total_quantity DESC;
